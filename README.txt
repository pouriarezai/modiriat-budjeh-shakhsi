Project Overview:

This project is a personal budget management system that helps users manage their income, expenses, and track their financial status. The system provides functionalities to add, update, and view income and expenses, and to generate summary reports.

Files Included:

 1. budget.py: Contains classes and methods for managing income and expenses, including adding, updating, and viewing financial transactions.
 2. database.db: SQLite database file used to store income and expense records.
 3. main.py: Provides a command-line interface to interact with the budget management system.

Usage:

 1. Run main.py to start the program.
 2. Follow the on-screen prompts to manage income, expenses, and view financial summaries.
